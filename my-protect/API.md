**1\. 获取登陆注册数据**

###### 接口功能
> 登陆

###### URL
> [/login/](/login/)

###### 支持格式
> JSON

###### HTTP请求方式
> post

###### 请求参数
> | 名称     | 字段类型 | 说明                    |
> | -------- | -------- | ----------------------- |
> | username | string   | 用户名称（11位）,唯一 |
> | password | string   | 密码（数字字母大小写6-25位）  |                  

###### 返回字段
> |返回字段|字段类型|说明                              |
> |:-----   |:------|:-----------------------------   |
> |code   |int    |0成功，1失败  |
> |msg|string |成功"success"，失败"fail" |
> | username | string   | 返回用户名 |
